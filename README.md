## Project Workflow

To ensure data accuracy and professional visualization, the following methodology was applied:

1. **Data Cleaning & Preparation:**
   - Organized raw data into structured tables.
   - Standardized data types and handled missing values to ensure consistency.

2. **Pivot Tables Construction:**
   - Created multiple Pivot Tables to analyze key metrics: (Employee Count, Total Salary, and Average Performance Rating).
   - Applied advanced "Value Field Settings" to calculate the **Average** for performance ratings instead of the default sum.

3. **Interactive Dashboard Design:**
   - Developed dynamic charts linked to **Slicers** (Department & Status) for real-time filtering and interactivity.
   - Cleaned the visual interface by removing gridlines and field buttons to enhance readability and professional aesthetics.

4. **Insights Extraction:**
   - Analyzed data trends to answer critical business questions regarding budget allocation and departmental efficiency.
